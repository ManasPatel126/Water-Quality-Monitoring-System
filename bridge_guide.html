One file. One line change. Two modes.

The sketch.ino runs in SIMULATION mode by default ‚Äî potentiometers pretend to be sensors. When your physical circuit is ready, uncomment #define PHYSICAL_MODE and re-upload. The same pins, same code, same serial output ‚Äî real sensors just replace the pots. This page explains exactly what swaps out.

The One-Line Switch

To go from simulation ‚Üí physical:

sketch.ino ‚Äî top of fileBEFORE (simulation)
// #define PHYSICAL_MODE ‚Üê this line is commented out ‚Üí simulation mode
sketch.ino ‚Äî top of fileAFTER (physical)
#define PHYSICAL_MODE ‚Üê remove the // ‚Üí physical mode. Re-upload. Done.
That's literally it. Nothing else changes. Same GPIO pins, same WHO thresholds, same serial report format, same LED indicators. The code automatically uses the correct reading formula for whichever mode is active.

Transition Steps ‚Äî Simulation to Physical
1

Keep everything from simulation ‚Äî only REPLACE the potentiometers

The ESP32 stays in the same position. The DS18B20, resistor, and LED wires stay exactly the same. The only things that change are the 3 potentiometers ‚Äî remove them and connect the real sensor modules to the same GPIO pins in their place.

2

Connect pH Module PCB where the pH potentiometer was

The pot had 3 wires: GND, SIG, VCC going to the breadboard rails and GPIO34. Replace with pH PCB: VCC‚Üí3.3V rail, GND‚ÜíGND rail, Po‚ÜíGPIO34. Lock the BNC glass probe into the PCB first.

Before powering on: Calibrate with pH 7 buffer ‚Äî adjust the blue trimmer pot on the PCB until Serial Monitor reads 7.00.
3

Connect TDS Module where the TDS potentiometer was

Same 3-wire swap: VCC‚Üí3.3V, GND‚ÜíGND, AOUT‚ÜíGPIO35. Plug the XH2.54 probe connector into the PCB header before wiring. Dip the 2-rod probe into the water sample.

4

Connect Turbidity Sensor where the turbidity potentiometer was

Same 3-wire swap: VCC‚Üí3.3V, GND‚ÜíGND, OUT‚ÜíGPIO32. If your module is 5V only, add voltage divider: 10kŒ© from sensor OUT to GPIO32, then 20kŒ© from GPIO32 to GND.

5

Open sketch.ino ‚Äî uncomment ONE line and re-upload

Find the line near the top: // #define PHYSICAL_MODE

Remove the // so it reads: #define PHYSICAL_MODE

Click Upload in Arduino IDE. Done. The Serial Monitor will now show "MODE: PHYSICAL (real sensors)" on boot.

6

Test each sensor before final deployment

  • Dip DS18B20 in water ‚Äî should read close to room temperature (20‚Äì28¬∞C)
  • Dip pH probe in tap water ‚Äî should read pH 6.5‚Äì8.0
  • Dip TDS probe in tap water ‚Äî should read 100‚Äì400 ppm
  • Dip turbidity sensor in clear water ‚Äî should read <2 NTU. Stir in a pinch of flour ‚Äî NTU should rise sharply.
  • LEDs: Green = safe, Yellow = 1 issue, Red = 2+ issues
7

Switch to battery power for portable deployment

Connect TP4056 OUT+ ‚Üí ESP32 VIN (not 3.3V!), TP4056 OUT‚àí ‚Üí ESP32 GND. Insert 18650 cell into holder. Unplug USB. The circuit should keep running from battery.

Battery life: ~12 hours continuous at 240mA. Use deep sleep between readings to extend to days/weeks.
